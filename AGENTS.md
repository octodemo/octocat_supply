# AGENTS

This project uses multiple AI agents and automation tools for code review, testing, and deployment. Please refer to this file for agent responsibilities and integration details.

## Agents
- **GitHub Copilot**: Code suggestions, refactoring, and documentation.
- **Claude**: Natural language understanding, advanced code review, and documentation generation.
- **Playwright**: End-to-end testing for the frontend.
- **Context7**: Contextual code analysis and search.
- **Atlassian MCP**: Issue and project management integration.

## Guidelines
- Use agents according to their strengths.
- Document any agent-specific configuration in this file.
- Update this file when adding or removing agents.
