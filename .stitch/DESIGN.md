# OctoCAT Supply - Product Search & Filter Design System

## Creative North Star: "The Vibrant Discovery Engine"
This design system creates a BOLD visual contrast between the calm, trusted e-commerce brand (green/white) and the exciting, high-energy search & filter feature. The filter UI should feel like a premium discovery tool that POPS off the page.

## Stitch Project
- **Project ID:** 13930509270135646389
- **Design System Asset ID:** 12746298771707689660

## Colors
### Brand Foundation
- **Primary Green (#76B852):** Brand identity, navigation, Add to Cart buttons, price text
- **White (#FFFFFF):** Card backgrounds, clean surfaces
- **Light Gray (#F3F4F6):** Page background

### SEARCH & FILTER FEATURE - Exciting Accent Colors
- **Electric Violet (#8B5CF6):** Filter panel controls, sort dropdown borders, active search ring, filter toggle button when active
- **Hot Coral (#FF6B6B):** Active filter count badges, Clear Filters button, filter removal X icons
- **Fuchsia Pink (#D946EF):** Gradient accent with violet for filter panel header strip
- **Gradient:** Linear gradient from Electric Violet (#8B5CF6) to Fuchsia Pink (#D946EF) for the filter panel top border or accent strip

### Surface Hierarchy
- Page background: #F3F4F6
- Product cards: #FFFFFF with subtle shadow
- Filter panel: #FFFFFF with a 3px top border gradient (violet to fuchsia)
- Active filter badges: Coral red pills with white text

## Typography
- All Inter font family
- Product titles: 600 weight, dark gray
- Prices: 700 weight, Primary Green
- Filter labels: 500 weight, medium gray

## Components

### Search Bar
- Full width, rounded-lg (8px), subtle border
- On focus: 2px Electric Violet ring with violet glow

### Filter Toggle Button
- Default: White bg, gray border, gray icon
- Active/Open: Electric Violet bg, white text, white icon
- Active filter indicator: Hot Coral badge with count

### Filter Panel
- White background card
- 3px top border: gradient from #8B5CF6 to #D946EF
- Soft shadow (0 4px 20px rgba(139,92,246,0.15))
- Price inputs: Rounded, violet focus ring
- Sort dropdown: Rounded, violet focus ring
- Clear Filters: Hot Coral text, no background, bold

### Product Cards
- White bg, 8px rounded, hover scale + green glow shadow
- Discount badges: Primary Green rotated ribbon
- Price: Primary Green, bold
- Add to Cart: Green bg when active, gray when disabled
